Atribut ierarhic pe clasa

Sumar

Dacă o clasa ar avea un potențial de ierarhie, putem decora acea clasa cu atributul Q.Object.Attributes.HierarchicalObject, iar în results, în loc de un grid, va fi folosit un element de tip ierarhic.

Atributul HeirarchicalObject are urmatoare proprietăți:

  • ImplementationClass
  • CookieName
  • InitialExpand
  • DropConfirmMessage

În exemplul de mai jos, clasa a fost decorată cu atributul de ierarhie.

[Q.Object.Attributes.HierarchicalObject(
   ImplementationClass = typeof(Q.Object.DefaultHierarchicalObjectImpl), 
   CookieName="CRM_Objects_ContactCommercialCategory",
   InitialExpand = Q.Object.Attributes.HierarchicalObject
                       .InitialExpandMode.CookieBased,
   DropConfirmMessage="Do you confirm the move operation ?")]

Exemplu NewsCategory

Clasa NewsCategory conține o referință către ea însăși.

[Q.Object.Attributes.AdditionalLookup()]
static public Q.Object.Lookup.Lookup AdditionalLookup()
{
     Q.Object.Lookup.Lookup lookup = new Q.Object.Lookup.Lookup(Type);
     if (Foundation.Etc.Utility.HideSA)
          lookup.AddFilter(Q.Object.Lookup.Filter.OpNotFullLike, "sa", Elements.Login);

     return lookup;
}

Unde, NewsCategroyHierarchicalObjectImpl este o clasă ce moștenește Q.Object.DefaultHierarchicalObjectImpl și suprascrie metode precum GetInitialLookup, GetNodeDescription, GetNodeIcon.

Implementare cu provider custom

Implementarea cu provider custom poate fi folosit extinzând clasa Q.Object.DefaultHierarchicalObjectImpl, unde pot fi suprascrise funcționalități, cum ar fi : GetInitialLookup, GetNodeDescription, etc.

Exemplu

În următorul exemplu avem un provider custom unde metodele GetInitialLookup și GetNodeDescription au fost suprascrise.

public class CatalogFolderHierarchicalObjectImpl : Q.Object.DefaultHierarchicalObjectImpl
{
     public override Q.Object.Lookup.Lookup GetInitialLookup(Type objectType)
     {
          Q.Object.Lookup.Lookup lookup = base.GetInitialLookup(objectType);
          lookup.AddFilter(true, Elements.Enabled);
          return lookup;
     }

     public override String GetNodeDescription(Q.Object.Object instance)
     {
          CatalogFolder folder = instance as CatalogFolder;

          if (!folder.IsNew() && folder.OverrideVisibilityPolicy)
               return @" <i class=""icon-lock""></i> " + base.GetNodeDescription(instance);

          return base.GetNodeDescription(instance);
     }
}
