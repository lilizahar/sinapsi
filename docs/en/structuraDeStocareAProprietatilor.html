Structura de stocare a proprietatilor

Sumar

Proprietățile unei instanțe sunt stocate în baza de date dacă au fost decorate cu atributul Q.Object.Attributes.Property. Coloana este stocată în baza de date în coloana QDUS_PropertyName(ElementName), unde QDUS_Id este identity col. Se poate modifica explicit unde este stocată proprietate în baza de date dând o valoare proprietății ColumnName pentru atributul Q.Object.Attributes.Property.

Exemplu:

[Q.Object.Attributes.Property(ColumnName = "QDUS_UniqueIdentifier", Element = Elements.UniqueIdentifier)]
public String UniqueIdentifier2
{
    get
    {
        object value = DataGetImpl(Elements.UniqueIdentifier);
        if (value is String)
            return (String)value;
        else
            return ((Guid)value).ToString();
    }
    set
    {
        DataSetImpl(Elements.UniqueIdentifier, value);
    }
}

Instanta murdara

O instanță este considerata murdara daca una din proprietățile mapate în baza de date a fost alterată. Se poate verifica daca o instanta este murdara accesand ObjectStorage().Dirty. ObjectStorage(). Dirty poate fi utilizat doar în interiorul clasei care mosteneste Foundation.Objects.Object. Call-ul de update în baza de date se face doar atunci cand ObjectStorage().Dirty va avea valoare True. Se poate determina deasemea daca una dintre proprietati este murdara folosind metoda IsItemDirty din ObjectStorage().

Exemplu:

Boolean isFileDirty = ObjectStorage().IsItemDirty(Elements.File);

Valoare precedenta a unei proprietăți poate fi identificată înaintea salvării accesand InitialValue din ObjectStorage(). DirectAccess pasand din enum-ul proprietăților valoarea corespunzatoare.

Exemplu:

ObjectStorage().DirectAccess(Elements.User).InitialValue

Valoarea precedenta

Valoare precedenta a unei proprietăți poate fi identificată înaintea salvării accesand InitialValue din ObjectStorage(). DirectAccess pasand din enum-ul proprietatilor valoarea corespunzatoare.

Exemplu:

ObjectStorage().DirectAccess(Elements.User).InitialValue
