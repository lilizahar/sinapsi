Asocieri 1-1

Sumar

Asocierile de tip 1 - 1 între 2 obiecte se reprezinta printr-o entitate/tabela ajutătoare ce contine cate o referință către fiecare din cele 2 tipuri de obiect. De exemplu, un scscscNews poate avea o singura legătură cu un Media ( Media poate conține mai multe Media Items), iar la randul lui Media respectiv poate avea legatura doar cu acel News astfel avem definită o relație de tip 1 - 1.

Exemplu

Daca avem relatia:

N2N_RelationN2N_LeftN2N_Right
Local.Objects.News-WebParts.Objects.Media.Media-Media249

Nu poate exista nici o alta relație între același News (Id 2 - N2N_Left) și alt Media (diferit de Id 49 - N2N_Right ) și nici o alta relație între acel Media și alt News.

In clasa News vom avea:

[Q.Object.Attributes.One2One(Element = Elements.Media)]
public WebParts.Objects.Media.Media Media
{
     get
     {
          return GetOne2One<WebParts.Objects.Media.Media>(Elements.Media);
     }
     set
     {
          SetOne2One<WebParts.Objects.Media.Media>(Elements.Media, value);
     }
}

Relatiile 1 - 1 sunt salvate in tabela sinapsi.QD_N2N

Exemplu

Un News care are doua poze atașate la el. sinapsi.QD_N2N

N2N_RelationN2N_LeftN2N_Right
Local.Objects.News-WebParts.Objects.Media.Media-Media249

N2N_Relation N2N_Left N2N_Right Local.Objects.News-WebParts.Objects.Media.Media-Media 2 49

N2N_Relation:

  • Local.Objects.News - unde folosim : N2N_Left
  • WebParts.Objects.Media.Media- de unde folosim: N2N_Right
  • Media - ce folosim

Folosim în obiectul cu Id-ul 2 ( coloana N2N_Left) din tabela sinapsi.L_News, Elementele cu Id- urile 72 și 77 (din tabela sinapsi.Q_W_MediaItem) din Media cu Id-ul 49 (coloana N2N_Right) din tabela sinapsi.Q_W_Media.

Interogări baza de date:

  • select * from sinapsi.QD_N2N

  • select * from sinapsi.L_News

  • select * from sinapsi.Q_W_Media

  • select * from sinapsi.Q_W_MediaItem

  • select * from sinapsi.Q_W_MediaItemType

Rezultate:

public class Model : Foundation.MVC.Model
{
     protected override void ConfigureViews()
     {
         AddRootView();
          ((News)(Root)).LocalizedText.AddModelViews(this,                
                WebParts.Objects.LocalizedText.EditMode.All); //Localizare

         ((News)(Root)).Attachments.AddModelView(this, true); //Atasamente

           ((News)(Root)).Media.AddModelView(this, true); //Media
     }
}

Adăugăm în Model RootView:

  • View-ul pentru Localizare
  • View-ul pentru Atasamente
  • View-ul pentru Media
