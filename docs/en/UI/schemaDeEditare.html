Schema de editare

Sumar

Schemele de editare sunt declarate în partea UI a unei clase. O schema de editare este metodata statica care este decorata cu atributul Q.Object.Attributes.EditSchema și returnează un obiect de tipul Q.Object.UI.Controls.Edit.Schema. Dacă atributul EditSchema nu are setat proprietatea Name (Name = “Single” sau Name=”SingleWithLanguages”) atunci acea schema de editare va fi schema de editare predefinita.

Exemplu:

[Q.Object.Attributes.EditSchema()]
public static Q.Object.UI.Controls.Edit.Schema SchemaEdit()
{
     Q.Object.UI.Controls.Edit.SchemaBuilder builder = new Q.Object.UI.Controls.Edit.SchemaBuilder(Type);

     builder.Add(Elements.Enabled, Q.Resources.Strings.Activ, Q.Resources.Common.ActiveUserNotes);
     builder.AddDropDownList(Elements.Role, Q.Resources.Common.UserRole);
     builder.Add(Elements.Login, String.Format("{0}", Q.Resources.Strings.Login)).SetMandatoryDisplayName();

     return builder.Schema;
}

Builder

Construcția unei scheme de editare se va face prin intermediul clasei de suport Q.Object.UI.Controls.Edit.SchemaBuilder. SchemaBuilder are diferite metoda de a adăuga elemente în schema de editare. Metoda Add va adăuga un control apropiat tipului proprietății.

Exemplu

builder.Add(Elements.Title, "Title");
builder.Add(Elements.Subtitle, "Subtitle");
builder.AddDropDownList(Elements.NewsCategory, "Category");

Tipuri de elemente de editare

Metoda Add va adăuga un control apropiat tipului proprietății, dar se poate preciza și explicit ce element va fi folosit pentru o proprietate. Declararea explicită a tipului de element poate fi făcută prin apelarea metodei apropiate controlului (cum ar fi pentru un dropdownlist, AddDropDownList) sau prin adaugarea directa a controlului specific.

Exemplu:

În următorul exemplu se va folosi declara explicit ce elemente vor fi folosite pentru proprietățile: Role, Email si Password. Pentru Role se va folosi metoda AddDropdownlist, iar pentru Emai si Password se vor adăuga controalele explicit.

builder.AddDropDownList(Elements.Role, Q.Resources.Common.UserRole);

Q.Object.UI.Controls.Edit.ItemEmail email = new     
       Q.Object.UI.Controls.Edit.ItemEmail(GetSchema(Type).Items[Elements.Email]);
email.DisplayName = Q.Resources.Strings.Email;
email.LimitText = 255;
builder.Add(email);

Q.Object.UI.Controls.Edit.ItemPassword itemPwd = new 
       Q.Object.UI.Controls.Edit.ItemPassword(GetSchema(Type).Items[Elements.Password]);
itemPwd.AlterContainer.cssHostClass = "uhpwd";
itemPwd.DisplayName = String.Format("{0}", Q.Resources.Strings.Password);
itemPwd.SetMandatoryDisplayName();
itemPwd.Quote = Q.Resources.Common.UserPasswordNotes;
builder.Add(itemPwd);

Exemplu:

Adăugarea controlului pentru încărcarea unei imagini:

Q.Object.UI.Controls.Edit.ItemComposite itemComposite = new Q.Object.UI.Controls.Edit.ItemComposite(Q.Resources.Strings.Image);

Q.Object.UI.Controls.Edit.ItemUpload imageItem = new Q.Object.UI.Controls.Edit.ItemUpload(GetSchema(Type).Items[Elements.Image]);

imageItem.UploadPrefix = "/writable/";
imageItem.UploadMaintainName = true;
imageItem.UploadFileName = "/writable/";
imageItem.SaveFileNameOnly = true;
itemComposite.Add(imageItem);
Q.Object.UI.Controls.Edit.ItemUploadInfo uploadInfo = new 
        Q.Object.UI.Controls.Edit.ItemUploadInfo(GetSchema(Type).Items[Elements.Image]);

uploadInfo.PathPrefix = "/writable/";
itemComposite.Add(uploadInfo);
builder.Add(itemComposite);

Evenimente

Cele mai importante evenimente de care putem tine cont într-o schema de editare sunt evenimentul de inițializare a unei scheme de editare și evenimentul de schimbare a unei proprietăți.

Evenimentul de inițializare

Evenimentul de inițializare este folosit cel mai des în a schimba comportamentul elementelor. Comportamentele care se schimbă des sunt vizibilitatea, readonly, additionallookup (pentru elementele care suporta additional lookup).

Sintaxa declarativă a evenimentului de inițializare

Q.Object.UI.Controls.Edit.Schema schema = builder.Schema;
schema.Initialize += OnSchemaEditInitialize;

Exemplu 1:

În următorul exemplu proprietatea Name va fi Readonly, iar proprietatea Code va fi ascunsă.

private static void OnSchemaEditInitialize(object sender, Q.Object.UI.Controls.Edit.InitializeArgs e)
{
    Questionnaire questionnaire = e.Instance as Questionnaire;
    if ( questionnaire.Document != null && questionnaire.Document.Versioned )
    {
        e.Schema.GetItemByProperty(Elements.Name).SetReadOnly();
        e.Schema.GetItemByProperty(Elements.Code).Visible = false;
    }
}

Exemplu 2:

În următorul exemplu pentru proprietatea ProductClass se seteaza un additional lookup.

private static void OnSchemaInitialize(object sender, Q.Object.UI.Controls.Edit.InitializeArgs e)
{
    QuestionnaireType questionnaireType = e.Instance as QuestionnaireType;

    if (questionnaireType.Department !=null)
    {
        ((Q.Object.UI.Controls.Base.IAdditionalLookup)e.Schema
            .GetItemByProperty(Elements.ProductsClass)).AdditionalLookup 
            = Q.Object.Lookup.Lookup.Get(PDCs.ProductsClass.Type).AddFilter(questionnaireType.Department,PDCs.ProductsClass.Elements.Department);
    }
}

Evenimentul de schimbare a unei proprietăți

Evenimentul de schimbare a unei proprietăți este folosit atunci cand, pentru o anumită proprietate, la schimbarea valorii, vrem sa modificăm valoarea unei alte proproprietati. De fiecare data când evenimentul de schimbare a unei proprietăți se declanșează , se va declanșa și evenimentul de inițializare schemei de editare.

Sintaxa declarativă a evenimentului de schimbare a unei proprietăți

Q.Object.UI.Controls.Edit.Schema schema = builder.Schema;
schema.Rules[Elements.Assestment].Changed += OnAssestmentChanged;

Exemplu:

În următorul exemplu pentru schimbarea proprietății Assestment se vor modifica și alte proprietăți ale instanței.

private static void OnAssestmentChanged(object sender, Q.Object.UI.Controls.Edit.RuleArgs e)
{
    TestCompetency testCompetency = e.Instance as TestCompetency;
    if (testCompetency.Assestment)
    {
        testCompetency.RequiresManualEvaluation = true;
        testCompetency.ShowCorrectAnswer = true;
        testCompetency.TestQuestionsNumber = 1;
        testCompetency.TestMaxQuestionsNumber = null;
    }
    else
    {
        testCompetency.RequiresManualEvaluation = false;
        testCompetency.ShowCorrectAnswer = true;
        testCompetency.TestQuestionsNumber = null;
        testCompetency.TestMaxQuestionsNumber = null;
    }
}

Valoarea precedenta

Valoarea precedenta a unei proprietăți inainte de modificare poate fi accesată din argumentul RuleArgs, chemand metoda GetPreviousValue.

private static void OnAssestmentChanged(object sender, Q.Object.UI.Controls.Edit.RuleArgs e)
{
    Boolean previous = e.GetPreviousValue<Boolean>(Elements.Assestment);
}
