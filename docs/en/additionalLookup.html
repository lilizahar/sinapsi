Additional Lookup

Sumar

Additional lookup poate fi setat pentru a prefiltra un obiect în contextele în care acel obiect este folosit (obiecte din baza de date pe care nu vreau să le utilizez). Pentru a declara un additional lookup pentru un obiect trebuie creată o metodata statică care returnează un Q.Object.Lookup.Lookup, iar metoda trebuie decorată cu Q.Object.Attributes.AdditionalLookup.

Exemplu 1

[Q.Object.Attributes.AdditionalLookup()]
static public Q.Object.Lookup.Lookup AdditionalLookup()
{
     Q.Object.Lookup.Lookup lookup = new Q.Object.Lookup.Lookup(Type);
     if (Foundation.Etc.Utility.HideSA)
          lookup.AddFilter(Q.Object.Lookup.Filter.OpNotFullLike, "sa", Elements.Login);

     return lookup;
}

Exemplu 2

[Q.Object.Attributes.AdditionalLookup()]
        static public Q.Object.Lookup.Lookup AdditionalLookup()
        {
            Q.Object.Lookup.Lookup lookup = new Q.Object.Lookup.Lookup(Type);
            lookup.AddFilter(true, Elements.Active);

            return lookup;
        }
