Suprascriere reprezentare textuala a instantei

Sumar

Suprascrierea reprezentarii textuale a instanței se va face prin suprascrierea metodei ToString() mostenita din clasa de baza System.Object. Reprezentarea textuala va fi folosita ulterior in partea de UI.

Proprietăți/Colecții noi

Proprietățile sau colecțiile noi trebuie definite sub un alt enum (ex. Extensions), iar în declararea proprietăților sau a colecțiilor trebuie declarat explicit în atributul care decoreaza proprietatea proprietate / colecția.

Exemplu proprietate:

[Q.Object.Attributes.Property(Element = Extensions.Description)]
public String Description
{
    get
    {
        return DataGetImpl<String>(Extensions.Description);
    }
    set
    {
        DataSetImpl(Extensions.Description, value);
    }
}

Pentru o clasă Tag cu proprietatea nume se creează o clasă extinsă TagExtended ce are ca proprietate nouă description. În tabelul L_Tag se va adăuga această nouă proprietate description folosind query-ul:

alter table sinapsi.L_Tag
add LT_Description nvarchar(512)
go

De asemenea, folosim procedura AddObject care va adăuga obiectul în baza de date:

sinapsi.QD_AddObject 'Local.Objects.TagExtended'
go

Proprietățile Activate, Activate() și Type vor fi :

new public static TagExtended Activate(object identification)
{
    return Q.Object.ActivatorManager.Activator.ActivateObject<TagExtended>(identification);
}

[Q.Object.Attributes.Prototype()]
new public static TagExtended Activate()
{
    TagExtended prototype = Q.Object.ActivatorManager.Activator.ActivateObject<TagExtended>();
    return prototype;
}

new public static System.Type Type
{
    get
    {
        return typeof(TagExtended);
    }
}

Pentru partea de UI se va folosi atributul

[Foundation.Unit.Attributes.ModelBuilder(typeof(ModelExtended))]

și se va declara apoi un model extins ModelExtended.

Exemplu colectie:

[Q.Object.Attributes.Collection(Name = ExtendenCollection.Roles, CollectionType = typeof(Objects.Schema.UserRole), ReversePropertyName = Objects.Schema.UserRole.Elements.User)]
public Q.Object.CollectionBase Roles
{
    get
    {
        return DataGetCollectionImpl(Collection.Roles);
    }
}
